# Project Management System

A desktop-based **Project Management System** built in **Java** using Swing for the GUI and MySQL for the database.

## 📋 Overview

This is a simple yet functional Project Management System that allows users to:
- Login securely
- Add new projects
- View existing projects
- Remove projects
- Manage project details

## ✨ Features

- **User Authentication** (Login System)
- **CRUD Operations** for Projects
- **Clean Swing GUI** with icons
- **Database Integration** (MySQL)
- **Splash Screen** on startup
- **Responsive Dashboard** (Home Screen)

## 🛠️ Technologies Used

- **Java** (JDK)
- **Swing** (for GUI)
- **MySQL** (Database)
- **NetBeans IDE** (Project Structure)

🚀 How to Run

### Prerequisites
1. Install **JDK 8 or above**
2. Install **MySQL Server**
3. Install **NetBeans IDE** (recommended)
